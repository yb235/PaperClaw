# 📚 PaperClaw Documentation

Welcome to the PaperClaw documentation! This folder contains everything you need to understand how PaperClaw works — from the big picture down to every JSON field.

> **Who is this for?** Anyone — whether you're a new contributor, a curious developer, or someone who just wants to understand what this project does. We assume zero prior knowledge and explain everything from scratch.
